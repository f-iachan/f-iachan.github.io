# Assets Folder

Place the following files here:

1. `portrait.jpg` - Your professional photo (recommended: ~400x500px, <500KB)
2. `favicon.ico` - Site favicon (16x16 or 32x32 pixels)

## Getting a favicon

You can generate a simple favicon from your initials at:
- https://favicon.io/favicon-generator/
- Use FGV blue (#1C2F67) as the background color

## Photo recommendations

- Professional headshot
- Good lighting, neutral background
- Cropped to head and shoulders
- JPEG format, reasonable compression
